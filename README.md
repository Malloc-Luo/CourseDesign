### 一，创新

*   显示设定值和当前值—数码管
*   键盘设定温度
*   遥控控制温度
*   自制遥控器

*   A/D转换
*   串口通信及传输协议设置
*   定时器中断
*   温度控制算法

*   **读取温度**：A/D转换
*   ~~**风扇与加热设备控制**：PID算法~~
